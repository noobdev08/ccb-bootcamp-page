Web development

The Web Development track is designed to equip learners with the skills needed to build functional, responsive, and visually appealing websites and web applications. It covers both frontend development (what users see and interact with) and backend development (server-side logic and database integration).

What you will learn

  • HTML: Structure of web pages
  • CSS: Styling and layouts (Flexbox, Grid)
  • Responsive Design: Making websites mobile-friendly
  • JavaScript: Interactivity and logic
  • DOM Manipulation
  • Frameworks/Libraries: Introduction to React (or similar)

Tools and technologies

  • HTML
  • CSS
  • JavaScript
  • VS Code

Learning outcome

By the end of this track, you will have a solid understanding of web development fundamentals, be able to create responsive and interactive web pages, and have a portfolio of projects to showcase your skills to potential employers.

Duration: 3 Months
